Automated Fine Dust-Blocking Blinds

팀 B조 KKOY의 “미세먼지 차단 자동 블라인드” 프로젝트는 블라인드에 방충망과 필터 천을 결합하고, 미세먼지·온습도 센서로 실시간 데이터를 수집·시각화하여 기준 농도(나쁨) 이상 시 자동으로 블라인드를 제어하는 시스템입니다. 1~2주차에 부품 선정과 모형 제작을 완료했고, 3주차부터 센서·모터 회로 설계 및 아두이노 코딩을 병행했으며, 서보모터 개조로 360° 회전을 가능케 해 최종 주차에 프로토타입을 완성했습니다.

이 과정을 통해 저는 아두이노 코딩을 담당하며 기술적 한계에 부딪혔을 때 문제를 세분화하고, 팀원들과 협의하여 대체 방안을 실험해보는 융합적 사고의 중요성을 깨달았습니다. 일정 지연과 부품 수급 문제를 유연하게 관리하면서 프로젝트 리더십과 커뮤니케이션 역량도 크게 향상되었습니다. 또한, 실제 환경 데이터를 기반으로 자동 제어 작동 로직을 구현한 경험은 앞으로 스마트 홈·IoT 분야에서 보다 정교한 시스템을 설계하는 데 든든한 밑거름이 될 것입니다.
